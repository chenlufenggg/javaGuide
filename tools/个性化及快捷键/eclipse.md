## 快捷键

ctrl + shift + ↑(↓)    类中向上(下)定位方法

ctrl + shift + -([)    上下(左右)分屏

ctrl + shift + R    快速查找文件

ctrl + Q 定位到最后修改地方

ctrl + L 快速定位到某一行

ctrl + shift + 数字键盘/( *) 收起和展开代码
